# vue-app-start

> A Vue.js project

px demo: http://examples.itrydo.com/vue-app-start/px/index.html

rem demo: http://examples.itrydo.com/vue-app-start/rem/index.html

## Run

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev
```
